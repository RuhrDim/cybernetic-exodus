## Ключевые понятия

**ГКП** — Глобальный Кибернетический Планировщик: децентрализованная распределённая сеть управления ресурсами на основе BFT-протокола.

**Хаб** — минимальная автономная единица кибернетической цивилизации: самодостаточная структура на 500–2000 человек с производственным ядром, вертикальными фермами и медицинским центром.

**Репутационный капитал** — многомерный вектор подтверждённого вклада индивида, конвертируемый в приоритетный доступ к ресурсам сверх базового уровня.

**Атомарное распределение** — принцип логистики ГКП: перемещается не товар, а производственный файл; физически транспортируется только сырьё.
