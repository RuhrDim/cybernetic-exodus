## Über das Buch

„Der kybernetische Exodus" ist eine zweibändige Analyse der postkapitalistischen Zivilisation, die um das Konzept des **Globalen Kybernetischen Planers (GKP)** herum aufgebaut ist — eines dezentralen, verteilten Ressourcenverwaltungssystems auf Basis des BFT-Protokolls. Das Buch verbindet marxistische Wirtschaftsanalyse, Neurobiologie, Systemtheorie und politische Strategie.

Das Werk wird kostenlos verbreitet.

## Schlüsselbegriffe

| Begriff | Definition |
|---------|------------|
| **GKP** | Globaler Kybernetischer Planer — dezentrales, verteiltes Ressourcenverwaltungsnetz auf Basis des BFT-Protokolls |
| **Hub** | Kleinste autonome Einheit der kybernetischen Zivilisation (500–2000 Personen) mit Produktionskern, vertikalen Farmen und Medienzentrum |
| **BFT** | Byzantine Fault Tolerance — Ausfalltoleranzprotokoll; das System bleibt funktionsfähig, wenn bis zu einem Drittel der Knoten ausfällt |
| **Atomare Verteilung** | GKP-Logistikprinzip: Es wird eine Produktionsdatei bewegt, keine Ware; physisch transportiert werden nur Rohstoffe |
| **Reputationskapital** | Mehrdimensionaler Vektor des nachgewiesenen individuellen Beitrags, konvertierbar in vorrangigen Ressourcenzugang |
| **Präventive Homöostase** | GKP-Medizinprinzip: Beseitigung der physischen und umweltbedingten Krankheitsursachen vor ihrem Auftreten |
| **Neurosozium** | Form kollektiver Intelligenz auf Basis direkter kognitiver Integration (Gedankenexperiment, Zeithorizont: Jahrhunderte) |
| **OCC** | Organic Composition of Capital — organische Zusammensetzung des Kapitals (c/v) |
| **TRPF** | Tendency of the Rate of Profit to Fall — Tendenz der Profitrate zum Sinken |
| **Kognitive Rente** | Aneignung der Ergebnisse des kollektiven intellektuellen Menschheitserbes durch die Inhaber der Recheninfrastruktur |
| **Kognitiver Cashback** | Teilweise Rückführung angeeigneten Kollektivwissens in den gesellschaftlichen Kreislauf durch gemeinwohlorientierte KI-Nutzung |
